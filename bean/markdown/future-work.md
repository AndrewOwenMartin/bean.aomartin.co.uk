# Future Work

## Context Free SDS

In Context Free SDS, an agent in the diffusion phase selects a new hypothesis
if the agent is inactive and the polled agent is inactive (as per Passive
Diffusion) or if both the agent and the polled agent are active.

## Context Sensitive SDS

In Context Free SDS, an agent in the diffusion phase selects a new hypothesis
if the agent is inactive and the polled agent is inactive (as per Passive
Diffusion) or if both the agent and the polled agent are active and share the
same hypothesis.
