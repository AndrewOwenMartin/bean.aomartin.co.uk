## Bean the React app

Run dev server with
> cd bean
> npm run dev

To deploy
1. As amartin cd into bean.aomartin.co.uk/bean and run `./build.sh`
2. As root cd into bean.aomartin.co.uk and run `deploy.sh`

## sdsjs the TS library

`cd` into `sdsjs`.
Do `npm run $foo` where foo is one of the scripts listed in `package.json`.
